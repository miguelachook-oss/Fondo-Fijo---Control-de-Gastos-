📝 Descripción General
Esta aplicación está diseñada para darte el control total de tus fondos fijos, ingresos y egresos de forma rápida, segura y localmente. Permite registrar tus transacciones ordenadas por categorías, adjuntar copias digitales u fotografías de tus recibos físicos, analizar flujos de caja mediante reportes automáticos mensuales, y exportar la información a planillas de Excel con un acabado visual sumamente formal y profesional.

✨ Características Principales

Gestión de Fondo Fijo, Ingresos y Gastos:

Registro rápido de depósitos (entradas) y consumos (salidas).
Clasificación precisa por categorías para un mejor orden visual.

Reportes y Exportación Profesional (Excel)

Exporta tus movimientos a un documento de Excel formal e impecable que incluye:
Un encabezado elegante con la información del solicitante (Empleado, Cédula, Compañía, Teléfono).
Tarjetas de resumen (KPIs) de ingresos, egresos y saldo actual con alertas para saldos críticos.
Tabla de movimientos con diseño moderno de filas alternadas (cebra), indicadores visuales y soporte para fotos/vouchers digitales.
Espacio inferior alineado con firmas y líneas de aprobación formal.

Práctico Calendario Integrado:
Calendario interactivo mejorado con alto contraste visual que garantiza una lectura clara de los días seleccionados.

Multi-Divisa Regional:
Ajuste de monedas y divisas locales adaptadas para toda América y España.

Privacidad y Seguridad de Primer Nivel:
Protección de datos confidenciales del usuario mediante bloqueo con huella digital (biometría) o un código PIN confidencial.

Respaldos Automatizados:
Integración directa y segura con Google Drive para respaldar y recuperar tu base de datos local cuando lo necesites.
